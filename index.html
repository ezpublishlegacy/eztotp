EzTotp: Two-factor authentication for eZPublish

This software is in development and not stable nor fully functional yet! You use this at your own risk.

Description

This is an eZPublish extension which implements a two-factor authentication for the eZPublish login process.

The second authentication-factor is done with the Google Authenticatior mobile app: http://support.google.com/accounts/bin/answer.py?hl=en&answer=1066447

For more information refere to the SETUP and LICENSE file.
