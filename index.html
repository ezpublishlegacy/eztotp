EzTotp

An eZPublish Extension to enable two-factor authentication with Google Authenticator

Description

The EzTotp Extension enables a second authentication factor on your eZPublis Instance. The second authentication factor is a 6 digit TOTP key which is generated by the Google Authenticator mobile app. With this Extension, you are able to enable TOTP Authentication for some, or all your eZUsers.

Features

  • Two authentication factors for more login-security
  • Customizable authentication method and timespan
  • TOTP Login Interface for backend siteaccess
  • Advanced user management with enable and block function on eZPublis Admin2 Interface
  • User activation dialogue with QR-Code generator for Smartphone integration
  • Customizable Error- and Access-Logging on eZPublis Admin2 Interface
  • Self-documenting, unit-tested PHP5 OOP code.
  • Fancy admin GUI built on Twitter Bootstrap  and  jQuery
  • 100% Open Source

In a nutshell

All changes are inside the extension, there will be no changes made on your core eZPublish. EzTotp uses own persistant objects and database tables. Acces to other database-tables is read-only. If your want to remove the EzTotp Extension, delete the extension directory and the two "eztotp_" database-tables and your setup is as clean as before installation of EzTotp.
